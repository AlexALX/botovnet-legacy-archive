# Botov-Net Rust Server Archive (2015)

This directory contains the Rust game server setup, plugins, and tools
historically used for the Botov-Net project (botov.net.ua), which operated
until 2015.

It is published as a **historical archive and a reference for developers**
who want to study or reuse server configurations, plugins, and scripts
for their own projects.

> ⚠️ Note: All plugins were written for Rust version 2015.
> Without modifications, they are unlikely to work on modern Rust versions.

> ⚠️ Disclaimer: This project is a fan-made, archival project.
> The author has no affiliation with Facepunch Studios (Rust) or Valve (CS).

---

## Contents

- `rust-fun` - easy server setup what was used for Botov-NET
- `rust-normal` - normal server setup what was used for Botov-NET
- `plugins/` — custom plugins written
- `configs/` — server configuration files
- `plugins_list.txt` — list of plugins used on the server
- `for-web-playerlist/` — web script for store player information and display later

for your own projects, but **no game binaries or assets are included**.
