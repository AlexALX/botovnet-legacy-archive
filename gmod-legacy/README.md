# Botov-Net GMod Server Archive

This directory contains the Garry’s Mod server setup, plugins, and configurations
historically used for the Botov-NET project (botov.net.ua).

It is published as a **historical archive and a reference for developers**
who want to study or reuse server setups, addons, and plugins for their own projects.

> ⚠️ Disclaimer: This project is fan-made and archival.
> The author has no affiliation with Valve or Facepunch Studios.

---

## Contents

- `addons/` — server addons
  - `add_lua/` — custom Lua plugins written by AlexALX (MIT License except some)
  - `addon_list.txt` — list of addons used on the server
- `cfg/` and `data/` — server configuration files
- `maps-source/` - source code of stargate map what i did made in past (MIT License)

You can study, modify, and reuse the structure, configs, and plugins
for your own projects, but **no game binaries, maps, or addon assets are included**.
