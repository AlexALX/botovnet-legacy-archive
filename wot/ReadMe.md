# World of Tanks Files (2014)

## Overview

This folder contains files related to **World of Tanks (version 2014)**, used for clan mods and utilities.  

- `v1.2_md_net.iss` — custom mod installer made for an old clan in the game.  
- `WotKIEB.py` — source code of the mod version checker.  

> ⚠️ These files are provided **for reference only**.  
> They are **archival / historical** and will **not work** with modern versions of the game.  
> No warranties are provided; use at your own risk.

---

## Purpose

These files are included to:  

- Preserve old clan mod setups.  
- Provide **reference for historical or educational purposes**.  
- Allow others to explore how mod installers and updaters were structured in 2014.
